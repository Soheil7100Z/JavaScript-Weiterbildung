Übung 23: Fetch API – Einfache Datenabfrage

Erstelle eine Funktion fetchUsers, die mithilfe der Fetch API eine Liste von Benutzern von der DummyJSON-API abruft. Verwende async/await und try/catch, um Netzwerkfehler abzufangen. Die abgerufenen Daten sollen dann in der Konsole ausgegeben werden.

Hinweis: Verwende die URL https://dummyjson.com/users für den Fetch-Aufruf.
